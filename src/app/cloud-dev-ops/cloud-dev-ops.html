Microsoft Azure

Project - 01 : Azure Secure Application Hosting with VM Scale Sets & Load Balancer
  • Designed and implemented a secure, production-grade Azure application hosting architecture using Azure Virtual Networks, Subnets, NSGs, Load Balancer, and Virtual Machine Scale Sets.
  • Created a private Azure VNet with isolated subnets to ensure network segmentation and secure workload isolation.
  • Deployed Linux-based Virtual Machine Scale Sets to host a web application, enabling high availability and horizontal auto-scaling.
  • Configured a Standard Azure Load Balancer with backend pools, HTTP health probes, and load balancing rules to distribute traffic across VM instances.
  • Implemented Network Security Groups (NSGs) to allow inbound traffic only from the Azure Load Balancer on port 80, blocking all direct internet access to VMs.
  • Ensured zero public IP exposure on virtual machines, following Azure security and defense-indepth best practices.
  • Configured and linked an Azure Private DNS Zone to the VNet to enable internal name resolution using private DNS records.
  • Validated application resiliency, load distribution, and instance health using Azure Load Balancer health probes.
  • Demonstrated key cloud architecture principles including scalability, high availability, secure networking, and controlled access.
